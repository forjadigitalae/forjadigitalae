Política de Gestión de Datos

Recopilación de Datos ForjaDigitalAE recopila y procesa datos personales únicamente para fines legítimos relacionados con nuestros servicios de consultoría empresarial. Tipos de datos que procesamos: Datos de identificación: Nombre, email, teléfono, empresa Datos empresariales: Información sobre su empresa para evaluaciones Datos de navegación: Cookies técnicas para mejorar la experiencia Base legal para el procesamiento: Consentimiento explícito para evaluaciones y comunicaciones Interés legítimo para mejorar nuestros servicios Cumplimiento contractual para prestación de servicios Retención de datos: Conservamos sus datos personales solo durante el tiempo necesario para cumplir con los fines para los cuales fueron recopilados, o según lo requiera la ley colombiana. Transferencia de datos: No vendemos, alquilamos ni compartimos su información personal con terceros, excepto cuando sea necesario para proporcionar nuestros servicios o cuando lo requiera la ley.
